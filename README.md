# BlogHub - Blog Management App

A modern, responsive Blog Management application built with **React**, **Tailwind CSS**, and **TypeScript**.

## Features
- Create, view, and manage blog posts easily.
- Modern UI with Tailwind CSS for a seamless user experience.
- Fully responsive design for mobile and desktop.
